Shoe Catalog

Shoe Catalog is an E-Commerce like Web Application developed using ASP.NET Core MVC.

The following are the features of the App.
  • * CRUD Operations.
Tech used:
  • * ASP.NET Core MVC 7
  • * Entity Framework Core 7
  • * MS SQL Server
Design Pattern:
  • * Repository Pattern
  • * Unit Of Work
Future Implemenation:
  • * Payment Method
  • * Convert the Frontend side to ReactJS
